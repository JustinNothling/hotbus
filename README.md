# ⚡ HotBus

**Switchboard schematics in seconds, not days**

Upload your load schedule. Get a professional single-line diagram instantly.

## Features

- 🚀 **80% faster than manual CAD** - Generate schematics instantly from CSV data
- 📋 **IEC 60617 compliant** - Professional symbols and layouts that meet international standards
- 💰 **No AutoCAD license needed** - Generate professional schematics without expensive CAD software
- 🎯 **Professional design** - Dark theme with electric blue accents, built for engineers
- 📱 **Interactive viewer** - Zoom, pan, and download your schematics as SVG or PDF

## How it works

1. **Upload CSV** - Upload your circuit schedule in standard format
2. **Generate** - Our engine creates a professional single-line diagram following IEC 60617 standards
3. **Download** - Download your schematic as SVG or PDF, ready for CAD or direct use

## Tech Stack

- **Next.js 15** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Client-side generation** - No backend needed
- **Professional schematic engine** - Ported from Python to TypeScript

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to see the app.
